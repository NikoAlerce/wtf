Installation Instructions

This application requires Node.js with the following dependencies:

  • express
  • cors
  • @taquito/taquito
  • dotenv

To run the application:

  1. Install Node.js if not already installed
  2. Run npm install to install dependencies
  3. Copy .env.example to .env
  4. Run npm start to start the server
  5. Navigate to http://localhost:3000 in your browser

Note: It appears there might be an issue with npm or git installation on your system. Please ensure both are installed and configured correctly.
